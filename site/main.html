МОЙ ДОМ

Проект “Мой дом”

Чат-бот, в котором собрана вся необходимая информация по общежитию студенческого городка.

Наша цель: создать информационную базу из локально-нормативных актов студенческого городка для последующего запуска чат-бота.

«Мой дом» — твой гид по общежитию!
